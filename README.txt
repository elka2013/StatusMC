== Opis ==

Wyświetla informacje odnośnie serwera bukkit poprzez JSONAPI.

Plugin wymaga działającego pluginu JsonAPI po stronie serwera bukkit.
http://dev.bukkit.org/server-mods/jsonapi/

You must install JsonAPI Plugin on bukkit server before use it.

== Instalacja ==

1. Wgraj plugin StatusMC do `/wp-content/plugins/` na swoim serwerze
2. Aktywuj plugin poprzez zakładkę 'Plugins' w menu WordPress
3. Umieść nowy widget w porządanym miejscu i skonfiguruj go do działania z serwerem.
